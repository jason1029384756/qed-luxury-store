# QED Luxury Fashion Store

A modern luxury fashion ecommerce frontend built with React, Tailwind CSS and Framer Motion.

## Setup

```bash
npm install
npm start
```

## Deployment

Deploy easily using Vercel or Netlify.


## Push to GitHub

In your project folder:

```bash
git init
git add .
git commit -m "Initial QED luxury store frontend"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/qed-luxury-store.git
git push -u origin main
```

## Deploy (1 click)

1. Go to https://vercel.com
2. New Project → Import GitHub repo → Deploy
